💡 어떤 버전을 사용해야 할까?
═══════════════════════════════════════════════════════════════════════════

✅ 완전 버전 (model_training_notebook.py)을 사용하세요:

  📌 처음 모델 학습하는 경우
  📌 프로젝트 문서화가 필요한 경우
  📌 모든 모델을 체계적으로 비교하고 싶은 경우
  📌 Feature Importance를 자세히 분석하고 싶은 경우
  📌 발표/리포트 작성용

✅ 간단 버전 (model_training_simple.py)을 사용하세요:

  📌 빠르게 결과를 확인하고 싶은 경우
  📌 하이퍼파라미터 튜닝 중 반복 실행이 필요한 경우
  📌 특정 모델 몇 개만 비교하고 싶은 경우
  📌 메모리/시간이 제한적인 경우
  📌 코드를 커스터마이징하기 쉽게 하고 싶은 경우

🚀 실행 예시
═══════════════════════════════════════════════════════════════════════════

완전 버전:
  1. Jupyter Notebook 실행
  2. model_training_notebook.py 열기
  3. 각 셀(# %%)을 순차적으로 실행
  4. 약 3-6분 후 모든 결과 확인

간단 버전:
  1. Jupyter Notebook 실행
  2. model_training_simple.py 열기
  3. 각 셀(# %%)을 순차적으로 실행
  4. 약 1-2분 후 핵심 결과 확인

🎯 추천 워크플로우
═══════════════════════════════════════════════════════════════════════════

1단계: 간단 버전으로 빠른 테스트
   → 데이터 확인, 기본 모델 성능 파악
   → 어떤 모델이 잘 작동하는지 빠르게 확인

2단계: 완전 버전으로 상세 분석
   → 최고 성능 모델 선택
   → Feature Importance 분석
   → 문서화 & 리포트 작성

3단계: 하이퍼파라미터 튜닝 (선택)
   → 간단 버전 수정하여 반복 실험
   → GridSearchCV, RandomizedSearchCV 추가

4단계: 최종 모델 저장 & Streamlit 통합
   → best_model_*.pkl 사용
   → Streamlit 앱에 로드
